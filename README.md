# GwentCard
Unity3D制作的巫师3狂猎中的昆特牌小游戏

## 项目介绍
基于学习的目的，仿照巫师3狂猎中的昆特牌做了一个小的Demo，基本实现了游戏机制和卡牌效果。因为是初学的缘故，还有很多小的Bug，但不影响整体，游戏收录了巫师3的开场动画和3首BGM。卡牌包含了除了史凯利特群岛卡组之外所有的昆特牌，希望大家能喜欢！
